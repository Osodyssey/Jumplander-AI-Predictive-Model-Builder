## 🚀 Overview
**Jumplander AI Predictive Model Builder** is a **mini AI engine in JavaScript** for building predictive models on datasets.  
It allows users to experiment with **linear regression, input-output predictions, and dynamic learning**.  

> Built entirely with **JumpLander Coder32b AI**, optimized for educational use, rapid prototyping, and clear visualization.

## 💡 Key Features
- Linear regression model for predictions  
- Train with custom datasets (JSON arrays)  
- Predict future values based on learned patterns  
- Modular, easy-to-extend codebase  
- Real-time training feedback
